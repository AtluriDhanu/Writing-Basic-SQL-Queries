# Writing Basic SQL Queries
It involves writing SQL queries to retrieve data from the library database using SELECT statements. It includes filtering with WHERE, combining conditions using AND/OR, pattern matching with LIKE, range queries using BETWEEN, sorting with ORDER BY, and limiting results with LIMIT.

## Features
- Retrieve data using basic SELECT statements
- Filter records using WHERE, AND, OR
- Search using pattern matching with LIKE
- Perform range-based queries with BETWEEN
- Sort results with ORDER BY
- Limit the number of rows returned using LIMIT

## Technologies Used
- MySQL
- MySQL Workbench

## Steps to run
- Open MySQL workbench and select the library database "Library_db".
- Write the SELECT queries into the editor.
- Run each query to view the output and understands how filters, sorting and limit works.
- Modify values or conditions if you want to test additional cases.
